# 3-Bit Binary Counter (Hardware Design)

## Description
This project is a hardware implementation of a 3-bit binary counter using digital logic circuits.

## Functionality
- Counts from 000 to 111 (0 to 7)
- Uses sequential logic design
- Demonstrates basic digital electronics concepts

## Tools / Hardware Used
- Digital Logic Hardware Components (Flip-Flops, Gates)
- Circuit Design Setup

## Learning Outcome
Understanding how binary counters work at the hardware level.
