# 3-Bit Binary Counter (Arduino Hardware Project)

## Description
This project is a hardware implementation of a 3-bit binary counter using Arduino and electronic components.

## Components Used
- Arduino Uno
- 3 LEDs
- Resistors
- Jumper wires

## How It Works
Each LED represents a bit (0 or 1).  
The Arduino counts from 0 to 7 in binary and displays the result using LEDs.

## Output
000 → 001 → 010 → 011 → 100 → 101 → 110 → 111

## Learning Outcome
- Understanding binary representation
- Basic Arduino programming
- Hardware circuit design
